# Практична робота 7 — Масиви (Варіант 5)

## 📌 Опис
Розроблено систему обліку витрат (Expense Tracker) з використанням масивів у JavaScript.

## 🔧 Функціонал

### Система витрат:
- Додавання витрат
- Видалення витрат
- Фільтрація за категорією
- Фільтрація за датою
- Загальна сума витрат
- Середня витрата
- Статистика по категоріях
- Топ-3 витрати

### Алгоритми:
- runningTotal (накопичена сума)
- movingAverage (ковзне середнє)
- twoSum (пошук двох чисел)
- maxSubarraySum (алгоритм Кадане)

## 🛠 Використано:
- JavaScript (ES6)
- Методи масивів: map, filter, reduce
- Spread оператор

## ▶️ Запуск
Відкрити файл `index.html` через Live Server
